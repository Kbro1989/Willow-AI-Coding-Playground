### Prerequisites
- Node.js 18+
- (Optional) Google Gemini API Key

### Quick Start

```bash
git clone https://github.com/Kbro1989/Willow-AI-Coding-Playground.git
cd antigravity-engine
npm install --legacy-peer-deps

```

```env

```
